# Exploratory Analysis of Two-Dimensional Gaussian Channels (CV-QKD)

This project performs exploratory data analysis (EDA) on subchannels derived from a bivariate Gaussian distribution, focusing on:

- Error probability (Ep)
- Subchannel capacity
- Efficiency per SNR
- Comparison between P and Q quadratures

## Technologies
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn

## Results
- EP vs. Capacity relationship
- Efficiency vs. SNR
- Subchannel structure per bit
